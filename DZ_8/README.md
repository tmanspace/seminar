### Задача 7

#### Добавление alias-a 

`alias ll='ls -la'` – так можно добавить alias

Чтобы сделать alias постоянным нужно добавить `alias ll='ls -la'` в `~/.bashrc` (в моем случае `~/.zshrc`)

Чтобы обновить нынешнее окружение нужно выполнить `source ~/.zshrc`
